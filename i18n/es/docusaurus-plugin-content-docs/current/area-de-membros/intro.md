# Imágenes - Dimensiones

Las dimensiones ayudan a proporcionar la mejor experiencia dentro del área de miembros. Algunas imágenes son
flexibles, es decir, pueden adaptarse a variaciones de tamaño sin recortarse (crop).

## Página de Login

Logo

```bash
Ancho: 200px
Alto: 200px
Ratio: 1/1

Obs: Imagen flexible, se adapta a tamaños diferenciados.
```
Fondo

```bash
Ancho: 1440px
Alto: 900px
Ratio: 10/16

Obs: La imagen puede sufrir crop.
```
