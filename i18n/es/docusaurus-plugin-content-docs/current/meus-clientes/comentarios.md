# Comentarios (Experimental)

El área **Comentarios** centraliza todos los comentarios realizados por los alumnos en las clases, permitiendo visualizar, responder y gestionar las interacciones en un solo lugar.

**Importante:** la funcionalidad de comentarios solo está disponible para productos del tipo **Curso Online**. Los productos de otros tipos (Mentoría Continua, Kanban, A.B./Reels) no tienen esta opción.

---

## **Visión General**

En la parte superior de la página, tres indicadores resumen la situación actual de los comentarios:

| Indicador | Descripción |
| ----- | ----- |
| **Total** | Cantidad total de comentarios en el período |
| **No Leídos** | Comentarios aún no visualizados |
| **Nuevos Hoy** | Comentarios recibidos en el día actual |

---

## **Configuración de Comentarios por Producto**

Haz clic en **"Configuración de Comentarios por Producto"** para expandir el panel y gestionar la configuración de cada producto de forma individual.

Para cada producto del tipo Curso Online, puedes configurar:

| Configuración | Descripción |
| ----- | ----- |
| **Comentarios** | Activa o desactiva los comentarios en el producto |
| **Visibilidad** | Define si los comentarios son **Públicos** (visibles para todos los alumnos) |
| **Respuestas de Alumnos** | Activa o desactiva la posibilidad de que los alumnos respondan comentarios |

Usa el campo de búsqueda para localizar rápidamente un producto específico.

---

## **Lista de Comentarios**

La tabla muestra todos los comentarios recibidos con las siguientes columnas:

| Columna | Descripción |
| ----- | ----- |
| **Alumno** | Nombre e indicación de comentario **Nuevo** cuando aún no ha sido leído |
| **Comentario** | Vista previa del texto enviado por el alumno |
| **Clase** | Nombre de la clase donde se realizó el comentario |
| **Producto** | Producto al que pertenece la clase |
| **Fecha** | Tiempo transcurrido desde el comentario |
| **Acciones** | Marcar como leído o ver detalles |

**Filtros disponibles:**

* **Todos los productos** — filtrar por producto específico  
* **Estado** — Todos, No leídos o Leídos  
* **Período** — filtrar por rango de tiempo (ej.: Último mes)  
* **Buscar por alumno o texto** — localiza comentarios específicos

**Otras acciones disponibles:**

* **Marcar todos como leídos** — marca todos los comentarios visibles como leídos de una vez  
* **Actualizar** — recarga la lista manualmente

---

## **Detalles del Comentario**

Al hacer clic en el ícono de visualizar (ojo) en cualquier comentario, se muestra una ventana de **Detalles del Comentario** con:

* Nombre y correo electrónico del alumno  
* Clase y producto relacionados  
* Fecha del comentario  
* Visibilidad del comentario (**Público**)  
* Texto completo del comentario  
* Lista de respuestas ya enviadas

**Cómo responder un comentario**

1. En la sección **Responder**, elige cómo deseas responder usando el toggle:  
   * **Desactivado**: responde como **tú mismo**  
   * **Activado**: responde como el **mentor principal (dueño de la cuenta)** — ideal cuando un miembro del equipo necesita interactuar con el alumno representando al mentor  
2. Escribe tu respuesta en el campo de texto  
3. Haz clic en **"Enviar respuesta"**

Una notificación de **"Respuesta enviada con éxito"** confirma el envío.

Para eliminar un comentario, haz clic en **"Eliminar comentario"** en la parte inferior de la ventana.
