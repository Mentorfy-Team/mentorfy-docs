## Introducción

La gestión eficaz de rutas de aprendizaje es fundamental para crear experiencias educativas atractivas y personalizadas. Esta guía completa explorará las funcionalidades y mejores prácticas para organizar y gestionar rutas de aprendizaje.

## 1. Fundamentos de la Gestión de Rutas

### 1.1 ¿Qué son las Rutas de Aprendizaje?

Las rutas de aprendizaje son secuencias organizadas de contenidos educativos que guían al alumno por un camino de aprendizaje estructurado. Permiten:

- Organizar contenidos en categorías lógicas
- Crear secuencias de aprendizaje progresivas
- Ofrecer experiencias personalizadas a los alumnos

### 1.2 Elementos Básicos de una Ruta

Las rutas están compuestas por:

- Productos/Mentorías: Los contenidos educativos individuales
- Subrutas: Agrupaciones menores dentro de una ruta principal
- Banners: Imágenes representativas para cada subruta
- Ordenamiento: La secuencia en que aparecen los elementos

## 2. Creando y Organizando Rutas

### 2.1 Creación de Rutas Principales

Para crear una nueva ruta:

1. Accede a la página de gestión de rutas
2. Haz clic en "Agregar Nueva Ruta"
3. Define un nombre para la ruta
4. Opcionalmente, agrega un banner representativo

### 2.2 Agregando Contenido a las Rutas

Para poblar tus rutas:

1. Identifica la ruta a la que deseas agregar contenido
2. Presiona el botón + para agregar contenido
3. Elige el tipo de contenido que deseas agregar
4. Usa las opciones de ordenamiento para definir la secuencia

### 2.3 Creando Estructuras Complejas

Para rutas más elaboradas:

1. Crea una ruta que será la principal
2. Crea una nueva ruta y cambia el tipo a subruta con el botón de engranaje
3. Agrega contenidos a la subruta
4. Repite el proceso para crear subrutas dentro de la subruta

Ejemplo de estructura compleja:
```
Ruta Principal: "Especial 2024"
    Subruta: "Abril"
        Producto 1
        Producto 2
    Subruta: "Mayo"
        Sub-subruta: "Semana 1"
            Producto 3
        Sub-subruta: "Semana 2"
            Producto 4
```

## 3. Configuraciones Avanzadas de Productos/Mentorías

### 3.1 Visibilidad y Acceso

Controla quién puede ver y acceder a tus contenidos:

- Visibilidad: Determina si un producto aparece en la vitrina para no suscriptores
- Bloqueo: Mantiene la visibilidad para no suscriptores, pero bloquea temporalmente el acceso a un producto. (El famoso "Próximamente")

### 3.2 Ordenamiento y Destaque

Personaliza la presentación de tus rutas:

1. Usa las flechas de ordenamiento para ajustar la secuencia de los elementos
2. Define manualmente la posición usando números (0, 1, 2, etc.)

### 3.3 Gestión de Subrutas

Para organizar subrutas:

1. Usa la opción "Selecciona una ruta" para mover una ruta dentro de otra
2. Para eliminar una subruta, haz clic en la "x" junto a la selección de ruta

## 4. Visualización y Experiencia del Usuario

### 4.1 La Vitrina de Aprendizaje

La vitrina es donde los alumnos interactúan con tus rutas:

- Las rutas principales aparecen como categorías
- Las subrutas aparecen como productos/mentorías dentro de la ruta principal
- Los banners representan visualmente cada subruta

### 4.2 Navegación del Alumno

Los alumnos pueden:

- Explorar rutas y subrutas como una estructura de carpetas
- Ver solo los productos a los que tienen acceso
- Interactuar con contenidos desbloqueados

## 5. Estrategias y Mejores Prácticas

### 5.1 Creando Experiencias Personalizadas

- Usa subrutas para crear itinerarios de aprendizaje específicos
- Posiciona el mismo contenido en diferentes contextos para diversos objetivos de aprendizaje

### 5.2 Gestión de Contenido Dinámico

- Utiliza el bloqueo temporal para contenidos en actualización
- Ajusta la visibilidad para controlar el acceso a contenidos exclusivos o de pago

### 5.3 Optimizando la Organización

- Mantén una estructura lógica e intuitiva
- Usa banners atractivos para representar cada ruta
- Revisa y ajusta regularmente el ordenamiento de los contenidos

## 6. Integraciones y Entrega de Contenido

### 6.1 Entrega Automática de Rutas

Es posible configurar la entrega automática de rutas completas:

1. Integra con pasarelas de pago
2. Configura la entrega de múltiples productos en una sola transacción
3. Para ello, accede al producto/mentoría principal y ve a la pestaña "Integraciones" para más detalles.

### 6.2 Personalización Basada en Ventas

- Crea rutas específicas para diferentes ofertas o paquetes
- Usa integraciones para entregar contenidos personalizados según las compras del alumno

## Conclusión

La gestión eficaz de rutas de aprendizaje es una herramienta poderosa para crear experiencias educativas ricas y personalizadas. Al dominar las técnicas de organización, configuración y entrega de contenido a través de rutas, educadores y creadores de contenido pueden ofrecer itinerarios de aprendizaje estructurados y atractivos.

Recuerda que la flexibilidad es la clave: experimenta con diferentes estructuras, ajusta según los comentarios de los alumnos y sigue optimizando tus rutas para brindar la mejor experiencia de aprendizaje posible.
