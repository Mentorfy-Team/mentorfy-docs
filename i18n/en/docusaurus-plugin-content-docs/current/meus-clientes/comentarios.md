# Comments (Experimental)

The **Comments** area centralizes all comments made by students in lessons, allowing you to view, reply, and manage interactions in one place.

**Important:** the comments feature is only available for products of the **Online Course** type. Products of other types (Continued Mentoring, Kanban, A.B./Reels) do not have this option.

---

## **Overview**

At the top of the page, three indicators summarize the current state of comments:

| Indicator | Description |
| ----- | ----- |
| **Total** | Total number of comments in the period |
| **Unread** | Comments not yet viewed |
| **New Today** | Comments received on the current day |

---

## **Comment Configuration per Product**

Click **"Comment Configuration per Product"** to expand the panel and manage the settings for each product individually.

For each Online Course product, you can configure:

| Setting | Description |
| ----- | ----- |
| **Comments** | Enables or disables comments on the product |
| **Visibility** | Defines whether comments are **Public** (visible to all students) |
| **Student Replies** | Enables or disables the ability for students to reply to comments |

Use the search field to quickly find a specific product.

---

## **Comments List**

The table displays all received comments with the following columns:

| Column | Description |
| ----- | ----- |
| **Student** | Name and **New** comment indicator when not yet read |
| **Comment** | Preview of the text sent by the student |
| **Lesson** | Name of the lesson where the comment was made |
| **Product** | Product to which the lesson belongs |
| **Date** | Time elapsed since the comment |
| **Actions** | Mark as read or view details |

**Available filters:**

* **All products** — filter by specific product  
* **Status** — All, Unread, or Read  
* **Period** — filter by time range (e.g.: Last month)  
* **Search by student or text** — finds specific comments

**Other available actions:**

* **Mark all as read** — marks all visible comments as read at once  
* **Refresh** — manually reloads the list

---

## **Comment Details**

Clicking the view icon (eye) on any comment opens a **Comment Details** window with:

* Student's name and email  
* Related lesson and product  
* Comment date  
* Comment visibility (**Public**)  
* Full comment text  
* List of replies already sent

**How to reply to a comment**

1. In the **Reply** section, choose how you want to reply using the toggle:  
   * **Disabled**: replies as **yourself**  
   * **Enabled**: replies as the **main mentor (account owner)** — ideal when a team member needs to interact with the student representing the mentor  
2. Write your reply in the text field  
3. Click **"Send reply"**

A **"Reply sent successfully"** notification confirms the submission.

To remove a comment, click **"Delete comment"** at the bottom of the window.
