# My Teams

The **My Teams** area allows you to organize mentors into groups, assign clients to each mentor, and define which products each team has access to.

**Administrators** can manage all products, students, and storefront, even those not assigned to them. Permissions can be accumulated.

---

## **Overview**

The main screen displays all created teams. Each team shows:

* Team name  
* Linked available products  
* List of mentors with name, email, and number of visible clients

---

## **Create a New Team**

Click **"+ New Team"** in the upper left corner. In the form, fill in:

**Team Name** Enter the name to identify the group.

* Example: *My Team*

Click **"Save"** to create the team.

---

## **Manage Team**

Clicking **"Manage Team"** next to any team opens a menu with the following options:

**Mentors**

* **Add Mentor** — adds a mentor to the team

**Products**

* **Add Products** — makes products available to the team

**Team**

* **Delete Team** — permanently removes the team

---

## **Add Mentor**

Clicking **"Add Mentor"** opens a form with the following fields:

* **Add existing mentor to the team** — option to link a mentor already registered on the platform  
* **Email** — mentor's email  
* **Name** — full name  
* **Phone (optional)**  
* **Password (optional)**

**Advanced Permissions** Define the mentor's access level on the platform. For each area, you can grant **View** and/or **Create** permissions:

| Area | Available Permissions |
| ----- | ----- |
| Meetings | View / Create |
| Clients | View / Create |
| Videos | View / Create |
| Integrations | View / Create |
| Profile | View / Create |
| Products | View / Create |
| Teams | View / Create |
| Events/Calendar | View / Create |
| Student Report | View / Create |
| Storefront Design | View / Create |

Check **"Select All (Admin)"** to grant full access to the mentor.

You can also apply a saved **Permissions Template** by clicking the corresponding button.

Click **"Save"** to add the mentor.

---

## **Add Products to the Team**

Clicking **"Add Products"** in the Manage Team menu opens a window displaying all available products with their name and type. Select the products the team will have access to and click **"Assign"**.

---

## **Actions per Mentor**

Clicking **"Options"** next to each mentor within the team, the following actions are available:

* **Edit Mentor** — changes the mentor's data and permissions  
* **Assign Clients** — selects which clients will be assigned to the mentor  
* **View Assignments** — displays clients already assigned to the mentor, with the option to filter by team and export as CSV  
* **Remove Assignment** — removes the clients assigned to the mentor  
* **Remove Mentor** — permanently removes the mentor from the team

---

## **Assign Clients to a Mentor**

Clicking **"Assign Clients"** opens a window displaying all available clients with their name and email. Select the desired clients using the checkboxes and click **"Assign"**.

The listed clients have access to the team's products. If a client is not found, verify that they have access to the team's product and that the team has the product available.

---

## **View Assignments**

The **View Assignments** window displays all clients assigned to the selected mentor, with the columns: Name, Team, and Mentor's Email.

**Available filters:**

* Search by name or email  
* Filter by team

Click **"Export CSV"** to download the assignment list.

---

## **Permissions Template**

Click **"Permissions Template"** on the main screen to create or edit reusable permission templates, speeding up the registration of new mentors.

**How to create a template:**

1. Select an existing template to edit, or leave it blank to create a new one  
2. Enter the **Template Title**  
3. Define the permissions for each area (View / Create)  
4. Click **"Save"**

The created templates are available for selection when adding new mentors.
