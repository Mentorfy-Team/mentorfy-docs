# Comentários (Experimental)

A área **Comentários** centraliza todos os comentários feitos pelos alunos nas aulas, permitindo visualizar, responder e gerenciar as interações em um só lugar.

**Importante:** a funcionalidade de comentários está disponível apenas para produtos do tipo **Curso Online**. Produtos de outros tipos (Mentoria Continuada, Kanban, A.B./Reels) não possuem essa opção.

---

## **Visão Geral**

No topo da página, três indicadores resumem a situação atual dos comentários:

| Indicador | Descrição |
| ----- | ----- |
| **Total** | Quantidade total de comentários no período |
| **Não Lidos** | Comentários ainda não visualizados |
| **Novos Hoje** | Comentários recebidos no dia atual |

---

## **Configuração de Comentários por Produto**

Clique em **"Configuração de Comentários por Produto"** para expandir o painel e gerenciar as configurações de cada produto individualmente.

Para cada produto do tipo Curso Online, você pode configurar:

| Configuração | Descrição |
| ----- | ----- |
| **Comentários** | Ativa ou desativa os comentários no produto |
| **Visibilidade** | Define se os comentários são **Públicos** (visíveis para todos os alunos) |
| **Respostas de Alunos** | Ativa ou desativa a possibilidade de alunos responderem comentários |

Use o campo de busca para localizar rapidamente um produto específico.

---

## **Lista de Comentários**

A tabela exibe todos os comentários recebidos com as seguintes colunas:

| Coluna | Descrição |
| ----- | ----- |
| **Aluno** | Nome e indicação de comentário **Novo** quando ainda não lido |
| **Comentário** | Prévia do texto enviado pelo aluno |
| **Aula** | Nome da aula onde o comentário foi feito |
| **Produto** | Produto ao qual a aula pertence |
| **Data** | Tempo decorrido desde o comentário |
| **Ações** | Marcar como lido ou visualizar detalhes |

**Filtros disponíveis:**

* **Todos os produtos** — filtra por produto específico  
* **Status** — Todos, Não lidos ou Lidos  
* **Período** — filtra por intervalo de tempo (ex: Último mês)  
* **Buscar por aluno ou texto** — localiza comentários específicos

**Outras ações disponíveis:**

* **Marcar todos como lidos** — marca todos os comentários visíveis como lidos de uma vez  
* **Atualizar** — recarrega a lista manualmente

---

## **Detalhes do Comentário**

Ao clicar no ícone de visualizar (olho) em qualquer comentário, uma janela de **Detalhes do Comentário** é exibida com:

* Nome e e-mail do aluno  
* Aula e produto relacionados  
* Data do comentário  
* Visibilidade do comentário (**Público**)  
* Texto completo do comentário  
* Lista de respostas já enviadas

**Como responder um comentário**

1. Na seção **Responder**, escolha como deseja responder usando o toggle:  
   * **Desativado**: responde como **você mesmo**  
   * **Ativado**: responde como o **mentor principal (dono da conta)** — ideal quando um membro do time precisa interagir com o aluno representando o mentor  
2. Escreva sua resposta no campo de texto  
3. Clique em **"Enviar resposta"**

Uma notificação de **"Resposta enviada com sucesso"** confirma o envio.

Para remover um comentário, clique em **"Excluir comentário"** na parte inferior da janela.
